Class ModelSettings

The container of Live2D model settings.

The structure of properties is the same as a raw model settings JSON, but all the keys are converted into camelCase.

example
let settings = new ModelSettings({
    "motions": {
        "tap_body": []
    }
}, 'path/to/my-model.model.json');

let tapBodyMotions = settings.motion.tapBody;
